Помогает молодым людям разобраться в страховании.

Реализованы три ситуации где может пригодиться страховка смена ситуации происходи по кнопке посмотреть другое
После ситуации идет пояснение что такое страховка и зачем она нужна
В конце есть чат с ии, где можно спросить любой вопрос по страхованию (по руту igs/chat)
    -вопрос векторизируеься в embedding с помощью локальной модели и затем передается в vectore_store
    -vectorestore ищет подходящую информацию в chromadb и затем передает ее в groq_client
    -groq_client строит системный промпт и делает запрос в qroq

Так же есть возможность добавлять документы в chromadb по руту igs/admin/add 

**ссылка на сайт:** https://huggingface.co/spaces/SheldonA/igs_app

## 📁 Структура проекта
- `backend/` — FastAPI + RAG система (chromadb + groq)
- `fronted/` — React фронтенд
-  `static/` — сборка фронтенда
